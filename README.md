## Descrição

O Convert é uma aplicação web simples e funcional que permite realizar a conversão de valores entre moedas estrangeiras (Dólar, Euro e Libra) para o Real Brasileiro (BRL).

Este projeto foi desenvolvido com foco educacional durante a Formação FullStack da Rocketseat, visando praticar a manipulação do DOM, tratamento de eventos e lógica de programação com JavaScript puro.

## O que o app faz?

A aplicação fornece uma interface intuitiva onde o usuário pode:

- Inserir um valor numérico (com bloqueio automático de caracteres alfabéticos).
- Selecionar a moeda de origem desejada.
- Visualizar em tempo real a cotação utilizada.
- Obter o resultado convertido e formatado no padrão de moeda brasileiro.

## Tecnologias e Ferramentas

O projeto foi construído utilizando tecnologias fundamentais da web:
- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização visual, incluindo fontes customizadas via Google Fonts e design responsivo.
- **JavaScript (Vanilla):** Lógica de conversão, regex para validação de input e manipulação de elementos da interface.

## Instruções de Instalação

**Pré-requisitos**
Para rodar este projeto, você só precisa de um navegador moderno. Não é necessário instalar Node.js ou bancos de dados, pois é uma aplicação 100% Front-End.

**Etapas**
1. Clone este repositório:
````
git clone https://github.com/seu-usuario/nome-do-repositorio.git
````
2. Navegue até a pasta do projeto:
````
cd projeto-convert
````
3. Abra o arquivo `index.html` em seu navegador.

## Instruções de Uso
1. Abra a aplicação no navegador.
2. No campo Valor, digite o montante que deseja converter.
3. No campo Moeda, selecione entre Dólar, Euro ou Libra.
4. Clique no botão Converter em reais.
5. O resultado aparecerá no rodapé da página.
